# filipenos.github.io

Página pessoal/profissional publicada via GitHub Pages.

## Objetivo

Apresentar perfil de engenharia, experiência, projetos selecionados e links públicos (LinkedIn/GitHub).

## Stack

- HTML estático
- CSS embutido
- JavaScript mínimo (data de atualização)
- Deploy por GitHub Pages (branch `main`)

## Estrutura

- `index.html`: landing page principal.

## Rodar localmente

Abra o arquivo direto no navegador:

```sh
open index.html
```

Ou suba um servidor local simples:

```sh
python3 -m http.server 8080
```

Depois acesse: `http://localhost:8080`.

## Publicação

Qualquer push em `main` publica a página em:

- `https://filipenos.github.io/`

## Manutenção rápida

- Atualize conteúdo técnico (work/projetos/stack) no `index.html`.
- Revise metatags SEO/Open Graph/Twitter ao alterar headline.
- Valide links externos após mudanças de perfil/usuário.
